# Comic Audio Narrator - User Guide

## Welcome to Comic Audio Narrator

Comic Audio Narrator transforms your favorite comic books into engaging audio narratives, making comics accessible to blind and visually impaired readers. Our AI-powered system analyzes comic panels and creates professional-quality audio descriptions with character-appropriate voices.

## Getting Started

### System Requirements

- **Web Browser**: Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+
- **Internet Connection**: Stable broadband connection recommended
- **Audio**: Speakers or headphones for audio playback
- **Storage**: Sufficient local storage for downloaded audio files

### First Time Setup

1. **Access the Application**
   - Open your web browser
   - Navigate to the Comic Audio Narrator website
   - No account creation required - start using immediately

2. **Browser Permissions**
   - Allow file access when prompted
   - Enable audio playback permissions
   - Consider allowing notifications for processing updates

### Interface Overview

The main interface consists of three primary sections:

- **Upload Area**: Where you upload comic files
- **Library**: Your collection of generated audio narratives
- **Player**: Audio playback controls and information

## Uploading Comics

### Supported File Types

- **PDF Files**: Standard comic book PDFs
- **EPUB Files**: Digital comic books in EPUB3 format
- **File Size Limit**: Maximum 50MB per file

### Upload Process

1. **Select Your Comic**
   - Click the "Upload Comic" button or drag and drop your file
   - Supported formats: PDF, EPUB
   - Maximum file size: 50MB

2. **File Validation**
   - The system automatically validates your file
   - You'll see a green checkmark for valid files
   - Error messages will appear for invalid files

3. **Processing Status**
   - Watch the progress bar during processing
   - Processing time varies based on comic length (typically 2-10 minutes)
   - You can continue browsing while processing occurs

4. **Processing Steps**
   - **Panel Extraction**: Identifying individual comic panels
   - **Character Analysis**: Recognizing characters and assigning voices
   - **Scene Detection**: Understanding settings and context
   - **Narrative Generation**: Creating audio descriptions
   - **Voice Synthesis**: Generating character-appropriate audio

### Upload Tips

- **File Quality**: Higher resolution PDFs produce better results
- **File Organization**: Use descriptive filenames for easy library management
- **Processing Time**: Longer comics take more time to process
- **Multiple Uploads**: You can upload multiple comics simultaneously

## Audio Playback

### Player Controls

The audio player provides full accessibility and control:

#### Basic Controls
- **Play/Pause**: Spacebar or click the play button
- **Skip Forward**: Right arrow key or click forward button (10 seconds)
- **Skip Backward**: Left arrow key or click backward button (10 seconds)
- **Volume Control**: Up/Down arrow keys or use volume slider
- **Mute**: M key or click mute button

#### Advanced Controls
- **Playback Speed**: Adjust from 0.5x to 2.0x speed
- **Chapter Navigation**: Jump between comic panels/scenes
- **Bookmark**: Save your current position
- **Loop**: Repeat current section or entire audio

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Spacebar | Play/Pause |
| → | Skip forward 10 seconds |
| ← | Skip backward 10 seconds |
| ↑ | Increase volume |
| ↓ | Decrease volume |
| M | Mute/Unmute |
| F | Toggle fullscreen player |
| 1-9 | Jump to 10%-90% of audio |
| 0 | Jump to beginning |

### Audio Features

- **Character Voices**: Each character has a unique, consistent voice
- **Scene Setting**: Detailed descriptions of locations and atmosphere
- **Action Narration**: Dynamic descriptions of movement and events
- **Dialogue Integration**: Natural integration of character speech
- **Professional Quality**: Follows audio description standards

## Managing Your Library

### Library Overview

Your library contains all generated audio narratives with the following information:

- **Comic Title**: Extracted from the original file
- **Duration**: Total audio length
- **Upload Date**: When you processed the comic
- **File Size**: Size of the audio file
- **Characters**: List of identified characters
- **Status**: Processing status or completion

### Library Features

#### Search and Filter
- **Search by Title**: Type in the search box to find specific comics
- **Filter by Date**: Show recent uploads or older content
- **Filter by Duration**: Find short or long audio narratives
- **Sort Options**: Alphabetical, date, or duration sorting

#### File Management
- **Download**: Save audio files to your device
- **Stream**: Play directly without downloading
- **Delete**: Remove unwanted audio narratives
- **Share**: Get shareable links for audio files

### Storage Options

#### Local Storage
- Audio files can be downloaded to your device
- Recommended for offline listening
- Manage storage space as needed

#### Cloud Library
- All audio narratives are stored in your cloud library
- Access from any device with internet connection
- Automatic backup and synchronization

## Accessibility Features

Comic Audio Narrator is designed with accessibility as a core principle:

### Screen Reader Support

- **Full Compatibility**: Works with NVDA, JAWS, VoiceOver, and other screen readers
- **Semantic HTML**: Proper heading structure and landmarks
- **ARIA Labels**: Comprehensive labeling for all interactive elements
- **Focus Management**: Logical tab order and focus indicators

### Keyboard Navigation

- **Full Keyboard Access**: All features accessible without a mouse
- **Standard Shortcuts**: Familiar keyboard patterns
- **Focus Indicators**: Clear visual focus indicators
- **Skip Links**: Quick navigation to main content areas

### Visual Accessibility

- **High Contrast**: Meets WCAG 2.2 AA contrast requirements (4.5:1 minimum)
- **Text Scaling**: Supports up to 200% text enlargement
- **Color Independence**: Information not conveyed by color alone
- **Clear Typography**: Readable fonts and appropriate sizing

### Audio Accessibility

- **Volume Control**: Granular volume adjustment
- **Speed Control**: Adjustable playback speed (0.5x to 2.0x)
- **Pause Anywhere**: Pause and resume at any point
- **Chapter Navigation**: Jump between sections easily

### Customization Options

- **Theme Selection**: Light and dark themes available
- **Font Size**: Adjustable text size
- **Audio Preferences**: Customize voice speed and volume defaults
- **Keyboard Shortcuts**: Customizable hotkeys

## Troubleshooting

### Common Issues and Solutions

#### Upload Problems

**Problem**: File won't upload
- **Solution**: Check file format (PDF or EPUB only)
- **Solution**: Ensure file size is under 50MB
- **Solution**: Try refreshing the page and uploading again

**Problem**: Upload fails during processing
- **Solution**: Check internet connection stability
- **Solution**: Try uploading during off-peak hours
- **Solution**: Contact support if problem persists

#### Playback Issues

**Problem**: Audio won't play
- **Solution**: Check browser audio permissions
- **Solution**: Ensure speakers/headphones are connected
- **Solution**: Try refreshing the page

**Problem**: Poor audio quality
- **Solution**: Check internet connection speed
- **Solution**: Try downloading for offline playback
- **Solution**: Adjust audio quality settings if available

#### Performance Issues

**Problem**: Slow loading or processing
- **Solution**: Close unnecessary browser tabs
- **Solution**: Check available storage space
- **Solution**: Try using a different browser

**Problem**: Interface not responding
- **Solution**: Refresh the page
- **Solution**: Clear browser cache and cookies
- **Solution**: Disable browser extensions temporarily

### Browser-Specific Issues

#### Chrome
- Enable hardware acceleration for better performance
- Check for browser updates regularly
- Clear cache if experiencing issues

#### Firefox
- Ensure media autoplay is enabled
- Check privacy settings for file access
- Update to latest version

#### Safari
- Enable JavaScript and media playback
- Check security settings for file uploads
- Clear website data if needed

## Supported Formats

### Comic File Formats

#### PDF Files
- **Standard PDFs**: Most comic book PDFs are supported
- **Scanned Comics**: High-quality scanned pages work best
- **Digital Comics**: Native digital comics provide optimal results
- **Resolution**: Higher resolution images produce better analysis

#### EPUB Files
- **EPUB3 Format**: Modern digital comic format
- **Fixed Layout**: Comics with fixed page layouts
- **Reflowable Content**: Some reflowable comics supported
- **DRM-Free**: Only DRM-free files are supported

### Quality Guidelines

For best results, use files with:
- **High Resolution**: 300 DPI or higher for scanned content
- **Clear Text**: Readable text in speech bubbles
- **Good Contrast**: Clear distinction between characters and backgrounds
- **Complete Pages**: All panels and pages intact

### Unsupported Formats

The following formats are not currently supported:
- CBR/CBZ comic archive files
- DRM-protected content
- Password-protected PDFs
- Corrupted or damaged files
