# Comic Audio Narrator - Troubleshooting Guide

## Quick Solutions

### Most Common Issues

| Problem | Quick Fix |
|---------|-----------|
| File won't upload | Check file format (PDF/EPUB only) and size (<50MB) |
| Audio won't play | Check browser permissions and audio settings |
| Slow processing | Check internet connection and try during off-peak hours |
| Interface not loading | Clear browser cache and refresh page |
| Keyboard shortcuts not working | Check for browser extension conflicts |

## Upload Issues

### File Upload Problems

#### Problem: "File format not supported"
**Symptoms:**
- Error message appears when selecting file
- Upload button remains disabled
- File is rejected immediately

**Solutions:**
1. **Check File Format**
   - Only PDF and EPUB files are supported
   - Ensure file extension is .pdf or .epub
   - Verify file is not corrupted

2. **File Conversion**
   - Convert CBR/CBZ files to PDF using comic readers
   - Use online converters for other formats
   - Ensure converted files maintain quality

3. **File Integrity**
   - Try opening file in another application
   - Re-download file if obtained online
   - Check for file corruption

#### Problem: "File too large"
**Symptoms:**
- Upload fails with size error
- Progress bar stops at 0%
- Browser becomes unresponsive

**Solutions:**
1. **Reduce File Size**
   - Maximum file size is 50MB
   - Compress PDF using PDF tools
   - Reduce image quality if necessary

2. **Split Large Comics**
   - Divide large comics into smaller sections
   - Process each section separately
   - Combine audio files if needed

3. **Alternative Formats**
   - Try EPUB format if available
   - Look for optimized versions of the comic

#### Problem: Upload starts but fails during processing
**Symptoms:**
- Upload begins successfully
- Progress bar stops partway through
- Error message appears after delay

**Solutions:**
1. **Check Internet Connection**
   - Ensure stable broadband connection
   - Avoid uploading on mobile data
   - Test connection speed (minimum 5 Mbps recommended)

2. **Browser Issues**
   - Try different browser (Chrome, Firefox, Safari)
   - Disable browser extensions temporarily
   - Clear browser cache and cookies

3. **Server Load**
   - Try uploading during off-peak hours
   - Wait and retry after 15-30 minutes
   - Check system status page for known issues

### Processing Issues

#### Problem: Processing takes too long
**Symptoms:**
- Processing exceeds expected time
- Progress appears stuck
- No updates for extended period

**Expected Processing Times:**
- Short comics (1-10 pages): 2-5 minutes
- Medium comics (11-30 pages): 5-15 minutes
- Long comics (31+ pages): 15-30 minutes

**Solutions:**
1. **Wait Patiently**
   - Complex comics take longer to process
   - AI analysis requires significant computation
   - Check progress periodically

2. **Check System Status**
   - Visit status page for service updates
   - Look for maintenance notifications
   - Check social media for announcements

3. **Contact Support**
   - If processing exceeds 1 hour
   - Provide file details and upload time
   - Include browser and system information

## Audio Playback Issues

### Audio Won't Play

#### Problem: No audio output
**Symptoms:**
- Play button doesn't respond
- Audio player shows error
- No sound from speakers/headphones

**Solutions:**
1. **Check Audio Hardware**
   - Verify speakers/headphones are connected
   - Test audio with other applications
   - Check system volume settings

2. **Browser Permissions**
   - Allow audio playback in browser settings
   - Check for blocked autoplay policies
   - Refresh page and try again

3. **Browser Compatibility**
   - Update browser to latest version
   - Try different browser
   - Check supported audio formats

#### Problem: Poor audio quality
**Symptoms:**
- Audio sounds distorted or unclear
- Frequent buffering or interruptions
- Low volume or muffled sound

**Solutions:**
1. **Internet Connection**
   - Check connection speed (minimum 1 Mbps for streaming)
   - Switch to wired connection if using WiFi
   - Close other bandwidth-intensive applications

2. **Audio Settings**
   - Adjust volume levels
   - Check audio quality settings
   - Try downloading for offline playback

3. **Hardware Issues**
   - Test with different headphones/speakers
   - Check audio drivers on computer
   - Adjust system audio settings

### Playback Control Issues

#### Problem: Keyboard shortcuts not working
**Symptoms:**
- Spacebar doesn't pause/play
- Arrow keys don't seek
- Volume keys don't work

**Solutions:**
1. **Focus Issues**
   - Click on audio player to focus
   - Ensure player has keyboard focus
   - Try Tab to navigate to player

2. **Browser Extensions**
   - Disable extensions temporarily
   - Check for keyboard shortcut conflicts
   - Try incognito/private browsing mode

3. **Browser Settings**
   - Check accessibility settings
   - Verify JavaScript is enabled
   - Clear browser cache

3. **Browser Compatibility**

   - Update browser to latest version

   - Use mouse to click then continue with keyboard

2. **Accessibility Mode**
   - Enable browser accessibility mode
   - Check high contrast settings
   - Verify focus indicators are visible

## Error Messages

#### "Network Error - Please check your connection"
**Cause:** Internet connectivity issues
**Solutions:**
- Check internet connection
- Try refreshing the page
- Switch to different network if available
- Contact ISP if connection problems persist

#### "Server Error - Please try again later"
**Cause:** Temporary server issues
**Solutions:**
- Wait 5-10 minutes and try again
- Check system status page
- Try during off-peak hours
- Contact support if error persists

#### "File Processing Failed"
**Cause:** Issues with comic file or processing pipeline
**Solutions:**
- Verify file is not corrupted
- Try different file format if available
- Check file meets size requirements
- Contact support with file details

#### "Audio Generation Failed"
**Cause:** Issues with text-to-speech processing
**Solutions:**
- Try processing again
- Check if comic has readable text
- Contact support if repeatedly fails
- Try with different comic file

#### "Storage Quota Exceeded"
**Cause:** Account storage limit reached
**Solutions:**
- Delete old audio files
- Download files and remove from cloud
- Contact support about storage limits
- Consider upgrading account if available

## Browser-Specific Issues

### Google Chrome

#### Common Chrome Issues
- **Hardware Acceleration**: Disable if causing graphics issues
- **Extensions**: Disable ad blockers and privacy extensions
- **Cache**: Clear cache more frequently due to heavy usage
- **Memory**: Chrome uses more memory - close unnecessary tabs

#### Chrome Solutions
```
Settings → Advanced → Reset and clean up → Clean up computer
Settings → Privacy and security → Clear browsing data
Settings → Advanced → System → Use hardware acceleration when available (toggle)
```

### Mozilla Firefox

#### Common Firefox Issues
- **Tracking Protection**: May block necessary scripts
- **Privacy Settings**: Strict settings can interfere
- **Add-ons**: Extensions may conflict with functionality

#### Firefox Solutions
```
Settings → Privacy & Security → Enhanced Tracking Protection (Standard)
Settings → Privacy & Security → Cookies and Site Data → Manage Exceptions
about:config → media.autoplay.default (set to 0)
```

### Safari

#### Common Safari Issues
- **Autoplay Policies**: Strict autoplay restrictions
- **Privacy Settings**: Intelligent Tracking Prevention
- **Cross-Site Tracking**: May block necessary requests

#### Safari Solutions
```
Safari → Preferences → Websites → Auto-Play (Allow All Auto-Play)
Safari → Preferences → Privacy → Prevent cross-site tracking (disable temporarily)
Develop → Disable Cross-Origin Restrictions (for testing)
```

### Microsoft Edge

#### Common Edge Issues
- **SmartScreen**: May block downloads
- **Tracking Prevention**: May interfere with functionality
- **Compatibility Mode**: May cause display issues

#### Edge Solutions
```
Settings → Privacy, search, and services → Tracking prevention (Balanced)
Settings → Downloads → SmartScreen for Microsoft Defender (adjust settings)
Settings → Default browser → Internet Explorer mode (disable)
```

#### iOS Solutions
- Use latest iOS version
- Close other apps to free memory

#### Common Android Problems
- **Browser Variations**: Different Android browsers behave differently
- **Memory Management**: Aggressive memory management may close tabs

#### Android Solutions
- Use Chrome or Firefox for best compatibility
- Keep app in foreground during processing
- Use "Request Desktop Site" if mobile version has issues
- Enable TalkBack for accessibility features
